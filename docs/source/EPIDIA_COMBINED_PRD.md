**4. EPIDIA Context Agent --- GitLab Duo Agent**

**4.1 What It Does**

The EPIDIA Context Agent is a GitLab Duo Agent that automatically
activates on GitLab issues and merge requests, reads project context and
deployment region metadata, queries EPIDIA\'s live global intelligence
APIs, and injects a structured real-world risk panel directly into the
GitLab UI.

It answers the question every developer should be asking but never is:
What is happening in the real world where this code will run?

**4.2 Agent Execution Flow**

1.  Developer opens or updates a GitLab Issue or Merge Request

2.  EPIDIA Context Agent triggers automatically via GitLab Duo Agent
    Platform

3.  Agent reads project description, deployment region tags, and code
    context

4.  Claude API synthesizes context and determines relevant intelligence
    modules to query

5.  Agent queries live APIs: FoodSight (FAO + NASA), ClimaRisk (ESA
    Copernicus), WealthFlow (World Bank)

6.  Claude API generates a structured risk intelligence summary with
    source attribution

7.  Agent injects a real-world context panel directly into the GitLab
    issue or MR

8.  Developer sees actionable risk signals and suggested responses ---
    without leaving GitLab

**5.4 Crisis Prediction Model**

The FoodSight crisis prediction model uses a weighted ensemble approach
across n real-time data streams:

P(crisis) = sum of \[w_i x f_i(x_t)\] + epsilon

Where w_i represents learned feature weights for each signal type,
f_i(x_t) represents the real-time value of signal i at time t, and
epsilon captures irreducible systemic uncertainty. Humility is built
into the model by design.

Validated accuracy on historical food crisis prediction: \>= 82% across
15 regional test cases spanning 2015-2024.
